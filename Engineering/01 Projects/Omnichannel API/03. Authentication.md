## Overview

The **Authentication Module** handles secure user authentication and credential management in the digital banking platform.

It manages certificate-based login, password reset workflows, password changes, and token lifecycle operations, integrating with Identity Adapter and Message Workers to ensure secure authentication, session management, and user notifications.
