## Overview

The **Dataset Module** provides centralized access to reference data and lookup services for the digital banking platform.

It manages static and semi-static datasets including geographical data (countries, states, cities), personal information options (genders, marital status, professions, relationships), time periods, and loan-specific intervals, integrating with Identity Adapter and CBA Adapter to ensure consistent data across the platform.
