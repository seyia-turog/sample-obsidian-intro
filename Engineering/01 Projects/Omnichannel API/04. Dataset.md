## Overview

The **Dataset Module** delivers standardized, centrally managed datasets used across the ADIBA ecosystem.  
It ensures consistent values for geographical data, personal information categories, scheduling periods, and loan intervals.

This module integrates directly with Identity and CBA adapters to fetch validated dataset records required across multiple business domains.
